# Planner

Calendario interativo editavel criado a partir de um projeto do Figma.

Projeto original: https://www.figma.com/design/VuRmNd4ad0rrOpltjNYch9/Calend%C3%A1rio-interativo-edit%C3%A1vel

## Como rodar

Instale as dependencias:

```bash
npm i
```

Inicie o servidor de desenvolvimento:

```bash
npm run dev
```

Gere o build de producao:

```bash
npm run build
```
